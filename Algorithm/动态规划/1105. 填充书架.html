1105. еЎ«е……д№Ұжһ¶ #

еҺҹйўҳең°еқҖпјҡhttps://leetcode.cn/problems/filling-bookcase-shelves/

йўҳи§Ј #

и®ҫdp[i]дёәе°ҶеүҚiжң¬д№Ұж”ҫе…Ҙд№Ұжһ¶зҡ„жңҖе°Ҹй«ҳеәҰпјҢз”ұдәҺеҸӘиғҪжҢүйЎәеәҸеҫҖйҮҢеЎ«е……пјҢжҲ‘д»¬еҸҜд»ҘиҖғиҷ‘еҰӮдёӢеҮ з§Қжғ…еҶөпјҡ

  • з¬¬iжң¬д№ҰеҚ•зӢ¬еҚ дёҖдёӘд№Ұжһ¶пјҢй«ҳеәҰзӯүдәҺеүҚi-1жң¬д№Ұзҡ„жңҖе°Ҹй«ҳеәҰеҠ дёҠз¬¬iжң¬д№ҰиҮӘе·ұзҡ„й«ҳеәҰ
    • иҝҷз§Қжғ…еҶөдёӢdp[i]=dp[i-1]+height[i]
  • е°ҶжңҖеҗҺдёҖеұӮзҡ„з¬¬i-1жң¬д№ҰжӢҝеҮәжқҘпјҢе’Ңз¬¬iжң¬д№Ұе…ұдёӨжң¬д№Ұз»„жҲҗдёҖеұӮд№Ұжһ¶пјҢжӯӨж—¶иҝҷеұӮд№Ұжһ¶зӯүдәҺиҝҷдёӨжң¬д№Ұй«ҳеәҰзҡ„жңҖеӨ§еҖј
    • иҝҷз§Қжғ…еҶөдёӢdp[i]=dp[i-2]+max(height[i],height[i-1])
  • д»ҘжӯӨзұ»жҺЁпјҢе°Ҷд»ҺеҗҺж•°з¬¬jжң¬д№ҰжӢҝеҮәжқҘпјҢе’Ңз¬¬iжң¬д№Ұе…ұj+1жң¬д№Ұз»„жҲҗдёҖеұӮд№Ұжһ¶пјҢжӯӨж—¶иҝҷеұӮд№Ұжһ¶зӯүдәҺиҝҷj+1жң¬д№Ұй«ҳеәҰзҡ„жңҖеӨ§еҖј
    • иҝҷз§Қжғ…еҶөдёӢdp[i]=dp[i-j-1]+max(height[i],...,height[i-j])

д»ҘжӯӨзұ»жҺЁпјҢдёҖзӣҙеҲ°иҝҷj+1жң¬д№Ұзҡ„жҖ»е®ҪеәҰ>д№Ұжһ¶е®ҪеәҰдёәжӯўпјҢе°ұжҳҜиҝҷеұӮд№Ұжһ¶иғҪеӨҹж»Ўи¶ізҡ„жүҖжңүжғ…еҶөпјҢжҲ‘д»¬еҜ№жүҖжңүжғ…еҶөеҸ–жңҖе°ҸеҖје°ұжҳҜд№Ұжһ¶зҡ„жңҖе°Ҹй«ҳеәҰпјҢжңүпјҡ

иҰҒжіЁж„Ҹзҡ„жҳҜпјҢз¬¬дёҖжң¬д№Ұзҡ„й«ҳеәҰheight[1]е№¶дёҚеҜ№еә”books[1]иҖҢжҳҜbooks[0]пјҢжүҖд»ҘеҜ№дәҺд»»ж„Ҹiжңү

иҫ№з•ҢжқЎд»¶пјҡ

еҲқе§ӢеҢ–dp[0]=0пјҢеҚіж”ҫе…Ҙ0жң¬д№Ұж—¶зҡ„й«ҳеәҰдёә0еҚіеҸҜпјҢжӯӨж—¶жңүdp[1]=dp[0]+height[1]=height[1]

ж—¶й—ҙеӨҚжқӮеәҰпјҡO(N^2)

з©әй—ҙеӨҚжқӮеәҰпјҡO(N)

java
class Solution {
    public int minHeightShelves(int[][] books, int shelfWidth) {
        int[] dp=new int[books.length+1];
        dp[0]=0;

        int widthSum,maxHeight,minHeight,totalHeight;
        for(int i=1;i<dp.length;i++){
            widthSum=0;
            minHeight=Integer.MAX_VALUE;
            maxHeight=Integer.MIN_VALUE;
            for (int j=0;j<i;j++){
                widthSum+=books[i-j-1][0];
                if(widthSum>shelfWidth)break;

                maxHeight=Math.max(maxHeight,books[i-j-1][1]);
                totalHeight=dp[i-j-1]+maxHeight;
                minHeight=Math.min(minHeight,totalHeight);
            }
            dp[i]=minHeight;
        }
        return dp[dp.length-1];
    }
}
