1143. жңҖй•ҝе…¬е…ұеӯҗеәҸеҲ— #

еҺҹйўҳең°еқҖпјҡhttps://leetcode.cn/problems/longest-common-subsequence

йўҳи§Ј #

жңҖй•ҝе…¬е…ұеӯҗеәҸеҲ—жҳҜдёҖйҒ“DPжЁЎжқҝйўҳпјҢйқўеҜ№иҝҷзұ»й—®йўҳжңүзӣёиҝ‘зҡ„жҖқи·Ҝ

ејҖдёҖдёӘдәҢз»ҙж•°з»„dpпјҢdp[i][j]иЎЁзӨәtext1.substring(0,i)е’Ңtext2.substring(0,j)зҡ„жңҖй•ҝе…¬е…ұеӯҗеәҸеҲ—

еҪ“жҲ‘д»¬йҒҚеҺҶеҲ°dp[i][j]ж—¶пјҢжңүдёӨз§Қжғ…еҶө

  • text1.charAt(i)==text2.charAt(j)
    • иҝҷз§Қжғ…еҶөеҸҜд»Ҙи®ӨдёәжҳҜдёӨдёӘеӯ—дёІXXXе’ҢXXXеҗ„еҫҖеүҚжү«жҸҸдәҶдёҖдҪҚпјҢз„¶еҗҺXXXAе’ҢXXXAжӯЈеҘҪеҸҜд»ҘеҢ№й…ҚдёҖдҪҚ
    • еӣ жӯӨdp[i][j]=dp[i-1][j-1]
  • text1.charAt(i)!=text2.charAt(j)
    • иҝҷз§Қжғ…еҶөXXXAе’ҢXXXBдёҚиғҪеҢ№й…ҚпјҢеӣ жӯӨжғ…еҶөзӯүеҗҢдәҺA/BжІЎжңүиў«жү«жҸҸеҲ°зҡ„жғ…еҶөпјҲе°ұеғҸжү“е®¶еҠ«иҲҚйҮҢзҡ„е°ҸеҒ·дёҖж ·пјҢеҰӮжһңдёҚеҒ·иҝҷй—ҙжҲҝеӯҗе®һйҷ…дёҠзәҰзӯүдәҺиҝҷй—ҙжҲҝеӯҗдёҚеӯҳеңЁ/жІЎжңүиў«йҒҚеҺҶеҲ°пјү
    • еӣ жӯӨdp[i][j]=max(dp[i-1][j-1],dp[i-1][j],dp[i][j-1])
    • з”ұдәҺйҖ’еҪ’жү«жҸҸзҡ„жғ…еҶөпјҲеңЁи®Ўз®—dp[i-1][j]е’Ңdp[i][j-1]зҡ„ж—¶еҖҷе·Із»ҸжҠҠdp[i-1][j-1]иҖғиҷ‘еңЁеҶ…дәҶпјүпјҢеӣ жӯӨdp[i-1][j-1]жҳҜеҸҜд»ҘзңҒз•ҘжҺүзҡ„
    • еӣ жӯӨdp[i][j]=max(dp[i-1][j],dp[i][j-1])

ж—¶й—ҙеӨҚжқӮеәҰпјҡO(MN)

з©әй—ҙеӨҚжқӮеәҰпјҡO(MN)

class Solution {
    public int longestCommonSubsequence(String text1, String text2) {
        int[][] dp=new int[text1.length()+1][text2.length()+1];
        for(int r=1;r<dp.length;r++){
            for(int c=1;c<dp[r].length;c++){
                if(text1.charAt(r-1)==text2.charAt(c-1))
                    dp[r][c]=dp[r-1][c-1]+1;
                else
                    dp[r][c]=Math.max(dp[r][c-1],dp[r-1][c]);
            }
        }
        return dp[dp.length-1][dp[0].length-1];
    }
}
