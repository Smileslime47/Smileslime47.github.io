# 70. 爬楼梯
原题地址：https://leetcode.cn/problems/climbing-stairs/

## 题解
设dp[i]为**爬到第i阶楼梯的可能性**，由于我们可以选择一次迈1-2步，我们可以从第i-1阶迈一步到i，也可以从i-2阶迈两步到i

于是我们有：

$$dp[i]=dp[i-1]+dp[i-2]$$

**边界条件**：dp[0]=dp[1]=1即可

**时间复杂度**：O(N)

**空间复杂度**：O(N)

```java
class Solution {
    public int climbStairs(int n) {
        int[] steps=new int[n+1];
        steps[0]=1;
        steps[1]=1;
        for (int i=2;i<=n;i++){
            steps[i]=steps[i-1]+steps[i-2];
        }
        return steps[n];
    }
}
```
