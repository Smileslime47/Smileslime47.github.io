6334. дёҖдёӘж•°з»„жүҖжңүеүҚзјҖзҡ„еҲҶж•° #

еҺҹйўҳең°еқҖпјҡhttps://leetcode.cn/problems/find-the-score-of-all-prefixes-of-an-array/

йўҳи§Ј #

йўҳзӣ®иҰҒжұӮиҝ”еӣһдёҖдёӘж•°з»„пјҢе…¶е®һжҲ‘д»¬зӣҙжҺҘзңӢжңҖеҗҺдёӨж®өпјҡans[i]жҳҜnums[0]вҖ”вҖ”nums[i]зҡ„converд№Ӣе’Ң

д№ҹеҸҜд»ҘеҶҷжҲҗпјҡans[i]=ans[i-1]+conver[i]

йӮЈд№Ҳе®һйҷ…дёҠconver[i]еҸӘжңүеңЁйҒҚеҺҶеҲ°iзҡ„ж—¶еҖҷжүҚдјҡз”ЁеҲ°пјҢд№ҹе°ұжІЎжңүеҝ…иҰҒејҖдёҖдёӘж•°з»„дәҶпјҢзӣҙжҺҘеңЁйҒҚеҺҶеҲ°iзҡ„ж—¶еҖҷи®Ўз®—iзҡ„converеҚіеҸҜ

жҜҸж¬ЎйҒҚеҺҶеҲҶдёүжӯҘпјҡ

  • з”ЁдёҖдёӘmaxз»ҙжҠӨnums[0]вҖ”вҖ”nums[i]зҡ„жңҖеӨ§еҖј
  • conver=max+nums[i]
  • ans[i]=ans[i-1]+conver

е®һйҷ…дёҠеҰӮжһңиҝӣдёҖжӯҘз®ҖеҢ–зҡ„иҜқпјҢеҸҜд»ҘзӣҙжҺҘеҶҷжҲҗans[i]=ans[i-1]+max+nums[i]

ж—¶й—ҙеӨҚжқӮеәҰпјҡO(N)

з©әй—ҙеӨҚжқӮеәҰпјҡO(N)

java
class Solution {
    public long[] findPrefixScore(int[] nums) {
        long[] ans=new long[nums.length];
        int max=0,conver;
        
        for(int i=0;i<nums.length;i++){
            if(max<nums[i])max=nums[i];

            conver=max+nums[i];

            if(i==0)ans[i]=conver;
            else ans[i]=ans[i-1]+conver;
        }
        return ans;
    }
}
