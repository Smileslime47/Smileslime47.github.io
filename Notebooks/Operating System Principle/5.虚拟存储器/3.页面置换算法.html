е…Ҳиҝӣе…ҲеҮәзҪ®жҚўз®—жі• #

йҖүжӢ©еңЁеҶ…еӯҳдёӯй©»з•ҷж—¶й—ҙжңҖд№…зҡ„йЎөйқўж·ҳжұ°

дҪҶдёҺе®һйҷ…иҝҗиЎҢи§„еҫӢдёҚз¬ҰпјҢеҰӮеңЁиҝӣзЁӢдёӯжңүдәӣз»Ҹеёёиў«и®ҝй—®зҡ„йЎөйқўпјҢеҗ«жңүе…ЁеұҖеҸҳйҮҸзӯү

жңҖиҝ‘жңҖд№…жңӘдҪҝз”Ёз®—жі•пјҲLRUпјү #

  • зЎ¬д»¶ж”ҜжҢҒпјҡ
    • жҜҸдёӘеҶ…еӯҳдёӯзҡ„йЎөйқўй…ҚзҪ®дёҖдёӘз§»дҪҚеҜ„еӯҳеҷЁ
      • и®ҝй—®жҹҗдёӘйЎөйқўж—¶е°Ҷе…¶еҜ„еӯҳеҷЁжңҖй«ҳдҪҚзҪ®1
      • жҜҸйҡ”дёҖж®өж—¶й—ҙе°ҶеҜ„еӯҳеҷЁеҸіз§»дёҖдҪҚ
      • еҜ„еӯҳеҷЁж•°еҖјжңҖе°Ҹзҡ„йЎөйқўеҚідёәжңҖиҝ‘жңҖд№…жңӘдҪҝз”ЁйЎөйқў

ClockзҪ®жҚўз®—жі• #

з®ҖеҚ•зҡ„Clockз®—жі• #

еҸҲз§°жңҖиҝ‘жңӘдҪҝз”Ёз®—жі•пјҲNRUпјү
иҝ‘дјјLRUз®—жі•пјҢдҪҶдёҚйңҖиҰҒиҫғеӨҡзЎ¬д»¶ж”ҜжҢҒ

  • дёәжҜҸдёӘйЎөйқўи®ҫзҪ®дёҖдҪҚи®ҝй—®дҪҚ
  • е°ҶеҶ…еӯҳдёӯжүҖжңүйЎөйқўйҖҡиҝҮжҢҮй’Ҳй“ҫжҺҘжҲҗдёҖдёӘеҫӘзҺҜйҳҹеҲ—
  • йҖҡиҝҮжҢҮй’ҲйҒҚеҺҶйҳҹеҲ—
    • иӢҘи®ҝй—®дҪҚдёә0еҲҷзҪ®жҚўиҜҘйЎөйқўз„¶еҗҺи®ҝй—®дҪҚзҪ®1
    • еҗҰеҲҷе°Ҷи®ҝй—®дҪҚ зҪ®0пјҢз„¶еҗҺжЈҖжҹҘдёӢдёҖйЎө

зјәйҷ·пјҡеҜ№дәҺеҶ…еӯҳдёӯжІЎжңүдҝ®ж”№иҝҮзҡ„йЎөйқўпјҢиҷҪз„¶жІЎжңүеҝ…иҰҒдҪҶзҪ®жҚўж—¶еҶҚж¬ЎеҶҷеӣһзЈҒзӣҳпјҢзҪ®жҚўд»Јд»·еӨ§

ж”№иҝӣеһӢClockз®—жі• #

  • дёәжҜҸдёӘйЎөйқўи®ҫзҪ®и®ҝй—®дҪҚAе’Ңдҝ®ж”№дҪҚM
    • AM=00ж—¶пјҢж—ўжңӘиў«и®ҝй—®д№ҹжңӘиў«дҝ®ж”№пјҢжҳҜжңҖдҪіж·ҳжұ°йЎө
    • AM=01ж—¶пјҢжңҖиҝ‘жңӘиў«и®ҝй—®дҪҶжӣҫз»Ҹиў«дҝ®ж”№иҝҮпјҢжҳҜж¬ЎдҪіж·ҳжұ°йЎө
    • AM=10ж—¶пјҢжңҖиҝ‘е·Іиў«и®ҝй—®дҪҶжІЎжңүиў«дҝ®ж”№пјҢеҸҜиғҪиў«еҶҚж¬Ўи®ҝй—®
    • AM=11ж—¶пјҢжңҖиҝ‘е·Іиў«и®ҝй—®дё”жӣҫз»Ҹиў«дҝ®ж”№пјҢеҸҜиғҪиў«еҶҚж¬Ўи®ҝй—®
  • жӯҘйӘӨ
    1. з¬¬дёҖиҪ®жү«жҸҸеҜ»жүҫж»Ўи¶іAM=00зҡ„з¬¬дёҖдёӘйЎөйқўпјҢжү«жҸҸжңҹй—ҙеҜ№Aе’ҢMдёҚиҝӣиЎҢдҝ®ж”№
    2. з¬¬дәҢиҪ®жү«жҸҸеҜ»жүҫж»Ўи¶іAM=01зҡ„з¬¬дёҖдёӘйЎөйқўпјҢе°ҶжӯӨеүҚжүҖжңүиў«жү«жҸҸиҝҮзҡ„йЎөйқўAеқҮзҪ®0
    3. иӢҘд»ҚжңӘжүҫеҲ°ж·ҳжұ°йЎөпјҢиҖҢжӯӨж—¶е·Іе°ҶжүҖжңүи®ҝй—®дҪҚAеқҮзҪ®0пјҢеӣһеҲ°з¬¬дёҖжӯҘйҮҚж–°жү«жҸҸпјҢдёҖе®ҡиғҪжүҫеҲ°ж·ҳжұ°йЎө

йЎөйқўзј“еҶІз®—жі•пјҲPage Buffering Algorithm a.k.a. PBAпјү #

еҪұе“ҚзҪ®жҚўзҡ„еӣ зҙ  #

  • йЎөйқўзҪ®жҚўз®—жі•
  • еҶҷеӣһзЈҒзӣҳзҡ„йў‘зҺҮ
  • иҜ»е…ҘеҶ…еӯҳзҡ„йў‘зҺҮ

з®—жі• #

еңЁеҶ…еӯҳдёӯи®ҫзҪ®дёӨдёӘй“ҫиЎЁ

  • з©әй—ІйЎөйқўй“ҫиЎЁ
    • з©әй—Ізү©зҗҶеқ—й“ҫиЎЁпјҢз”ЁдәҺеҲҶй…Қз»ҷйў‘з№ҒзјәйЎөзҡ„иҝӣзЁӢпјҢйҷҚдҪҺиҜҘиҝӣзЁӢзјәйЎөзҺҮ
    • еҪ“дёҖдёӘжңӘиў«дҝ®ж”№зҡ„йЎөиў«жҚўеҮәж—¶дёҚеҶҷеӣһзЈҒзӣҳпјҢиҖҢжҳҜж”ҫеңЁз©әй—Ій“ҫиЎЁзҡ„й“ҫе°ҫ
    • жІЎжңүиў«дҝ®ж”№зҡ„йЎөжІЎеҝ…иҰҒйҮҚж–°еҶҷеӣһзЈҒзӣҳпјҢеҲҶй…Қз»ҷж–°иҝӣзЁӢж—¶ж–°йЎөеҸҜд»ҘзӣҙжҺҘиҰҶзӣ–еҶҷе…ҘжӯӨйЎө
  • дҝ®ж”№йЎөйқўй“ҫиЎЁ
    • з”ұе·Ідҝ®ж”№йЎөйқўеҪўжҲҗзҡ„й“ҫиЎЁпјҢеҮҸе°‘е·Ідҝ®ж”№йЎөйқўжҚўеҮәзҡ„ж¬Ўж•°

и®ҝй—®еҶ…еӯҳзҡ„жңүж•Ҳж—¶й—ҙ #

иҜ·жұӮеҲҶйЎөз®ЎзҗҶж–№ејҸдёӯпјҢеҶ…еӯҳжңүж•Ҳи®ҝй—®ж—¶й—ҙпјҲEATпјүдёҚд»…иҰҒиҖғиҷ‘и®ҝй—®йЎөиЎЁе’Ңзү©зҗҶең°еқҖзҡ„ж—¶й—ҙпјҢиҝҳиҰҒиҖғиҷ‘зјәйЎөдёӯж–ӯзҡ„еӨ„зҗҶж—¶й—ҙ

  • О»дёәжҹҘжүҫеҝ«иЎЁж—¶й—ҙпјҢtдёәжҹҘжүҫеҶ…еӯҳж—¶й—ҙпјҢОөдёәдёӯж–ӯеӨ„зҗҶж—¶й—ҙ
    • иў«и®ҝй—®йЎөеңЁеҶ…еӯҳпјҢеҜ№еә”йЎөиЎЁйЎ№еңЁеҝ«иЎЁ
      • EAT=О»+t
    • иў«и®ҝй—®йЎөеңЁеҶ…еӯҳпјҢеҜ№еә”йЎөиЎЁйЎ№дёҚеңЁеқ—иЎЁ
      • EAT=2*(О»+t)
    • иў«и®ҝй—®йЎөдёҚеңЁеҶ…еӯҳ
      • EAT=2*(О»+t)+Оө
    • иҖғиҷ‘еҲ°еҝ«иЎЁе‘ҪдёӯзҺҮе’ҢзјәйЎөжғ…еҶөпјҢи®ҫaдёәе‘ҪдёӯзҺҮпјҢfдёәзјәйЎөзҺҮ
      • EAT=О»+at+(1-a)[t+f*(Оө+О»+t)+(1-f)*(О»+t)]
