1163. жҢүеӯ—е…ёеәҸжҺ’еңЁжңҖеҗҺзҡ„еӯҗдёІ #

еҺҹйўҳең°еқҖпјҡhttps://leetcode.cn/problems/last-substring-in-lexicographical-order/

йўҳи§Ј #

еҸӮиҖғе…¶д»–йўҳи§Ј

еҒҮи®ҫеӯҗдёІ[i,j]дёәеҪ“еүҚжңҖеӨ§еӯҗдёІпјҢиӢҘjдёҚдёәs.length()-1пјҢеҲҷеҝ…з„¶жңү[i,j+1]>[i,j]пјҢжүҖд»Ҙеӯ—з¬ҰдёІзҡ„еӯ—е…ёеәҸжңҖеӨ§еӯҗдёІеҝ…з„¶дёәsзҡ„дёҖдёӘеҗҺзјҖпјҢеҚіs.substring(i,s.length())

еҸҢжҢҮй’ҲпјҢи®ҫеүҚжҢҮй’ҲmaxдёәеҪ“еүҚжңҖеӨ§еӯҗдёІиө·е§ӢзӮ№пјҢеҗҺжҢҮй’ҲitrдёәеҪ“еүҚйҒҚеҺҶеӯҗдёІиө·е§ӢзӮ№пјҢkдёәеҪ“еүҚеӯҗдёІиў«жҜ”иҫғеӯ—з¬Ұзҡ„еҒҸз§»еҖј

жңҖеҲқиҖғиҷ‘еҲ°жҡҙеҠӣеҒҡжі•пјҢеҚіжҜҸж¬Ўmax++е’Ңitr++пјҢз„¶еҗҺйҮҚж–°йҖҗдёӘеӯ—з¬ҰжҜ”иҫғеӯҗдёІпјҢдҪҶжҳҜиҝҷж ·жңүдёӘй—®йўҳеңЁдәҺдјҡдёҚж–ӯжҜ”иҫғйҮҚеӨҚзҡ„йғЁеҲҶ

  • еҰӮabcdabcdbпјҢеңЁжҜ”иҫғе®Ңabcdaе’ҢabcdbеҗҺпјҢиҝҳдјҡжҜ”иҫғдёҖж¬Ўbcdaе’ҢbcdbпјҢз„¶еҗҺиҝҳдјҡжҜ”иҫғдёҖж¬Ўcdaе’Ңcdb...
  • иҝҷж ·дјҡеҜјиҮҙO(N^2)зҡ„ж—¶й—ҙеӨҚжқӮеәҰпјҢеңЁиҫғе°Ҹж•°жҚ®йҮҸж—¶иҝҳеҘҪиҜҙпјҢдҪҶжҳҜеҜ№дәҺ[a,a,...,a]иҝҷз§ҚеҲҒй’»ж ·дҫӢж—¶жҳҜеҝ…з„¶TLEзҡ„

е…ій”®еңЁдәҺеҰӮдҪ•е°ҶдёӨеұӮеҫӘзҺҜзј©еҮҸдёәдёҖеұӮеҫӘзҺҜпјҢи®©kжҜҸж¬ЎжҜ”иҫғдёҚдјҡжҜ”иҫғйҮҚеӨҚзҡ„йғЁеҲҶпјҢйўҳи§Јз»ҷеҮәзҡ„жҖқи·Ҝйқһеёёе·§еҰҷ

  • еҰӮжһңеҜ№дәҺдёӨдёӘеӯҗдёІж»Ўи¶і[max,max+k]=[itr,itr+k]зҡ„иҜқпјҢеҲҷеҸҜд»ҘзӣҙжҺҘеҫӘзҺҜk++зӣҙиҮіmax+kе’Ңitr+kеӯ—з¬ҰдёҚзӯү
  • еҰӮжһңеҜ№дәҺдёӨдёӘеӯҗдёІж»Ўи¶і[max,max+k]>[itr,itr+k]зҡ„иҜқ
    • жҲ‘д»¬еҫ—еҮәзҡ„з»“и®әдёҚеҸӘжҳҜеӯҗдёІ[max,end]>[itr,end]пјҢеӣ дёәиҝҷдёӘkзҡ„еӯҳеңЁпјҢеҜ№дәҺд»»дҪ•i<kжҲ‘д»¬йғҪжңүеӯҗдёІ[max+i,end]>[itr+i,end]пјҢеӣ дёә[max+i,max+k-1]дёҺ[itr+i,itr+k-1]иҝҷдёӨйғЁеҲҶе®Ңе…ЁзӣёзӯүпјҢиҖҢ[max+k]>[itr+k]пјҢжӯӨж—¶жңҖеӨ§еӯҗдёІеҝ…з„¶дёҚеңЁ[itr,end]вҖ”вҖ”[itr+k,end]дёӯпјҢжүҖд»ҘеҸҜд»ҘзӣҙжҺҘжҺЁиҝӣitr=itr+k+1
  • еҰӮжһңеҜ№дәҺдёӨдёӘеӯҗдёІж»Ўи¶і[max,max+k]<[itr,itr+k]зҡ„иҜқ
    • жҲ‘д»¬еҫ—еҮәзҡ„з»“и®әдёҚеҸӘжҳҜеӯҗдёІ[max,end]<[itr,end]пјҢеӣ дёәиҝҷдёӘkзҡ„еӯҳеңЁпјҢеҜ№дәҺд»»дҪ•i<kжҲ‘д»¬йғҪжңүеӯҗдёІ[max+i,end]<[itr+i,end]пјҢеӣ дёә[max+i,max+k-1]дёҺ[itr+i,itr+k-1]иҝҷдёӨйғЁеҲҶе®Ңе…ЁзӣёзӯүпјҢиҖҢ[max+k]<[itr+k]пјҢжӯӨж—¶жңҖеӨ§еӯҗдёІеҝ…з„¶дёҚеңЁ[max,end]вҖ”вҖ”[max+k,end]дёӯпјҢжүҖд»ҘеҸҜд»ҘзӣҙжҺҘжҺЁиҝӣmax=max+k+1
    • жӯӨж—¶иӢҘmax>=itrпјҢеҲҷиҜҙжҳҺitrд№ҹеңЁ[max,end]вҖ”вҖ”[max+k,end]дёӯпјҢжӯӨж—¶itrдёҚеӯҳеңЁжңҖеӨ§и§ЈпјҢеҸҜд»ҘзӣҙжҺҘйҮҚзҪ®itr=max+1

еңЁиҝҷз§Қжғ…еҶөдёӢпјҢеҜ№дәҺ[a,a,...,a]пјҢkдјҡз”ұдәҺmax+k==itr+kиҖҢдёҚж–ӯиҮӘеўһиҮіжң«е°ҫпјҢеңЁдёҖж¬ЎжҜ”иҫғе®ҢеҗҺitr+kдјҡзӣҙжҺҘеӣ дёәеӨ§дәҺs.length()иҖҢи·іеҮәеҫӘзҺҜпјҢжң¬иҙЁдёҠжҳҜеӣ дёәеҪ“жҲ‘д»¬жҜ”иҫғеҮәеүҚжҢҮй’Ҳе’ҢеҗҺжҢҮй’ҲдёӨдёӘеӯҗдёІзҡ„з»“жһңеҗҺпјҢеҜ№дәҺиҝҷдёӨдёӘеӯҗдёІзӣёеҗҢзҡ„йӮЈйғЁеҲҶеүҚзјҖд№ҹжҳҜе…ұдә«иҝҷдёӘз»“жһңзҡ„пјҢжүҖд»ҘдҝқиҜҒдәҶжҜҸж¬ЎеӯҗдёІжҜ”иҫғйғҪдёҚдјҡжңүйҮҚеӨҚжҜ”иҫғзҡ„йғЁеҲҶпјҢеңЁзәҝжҖ§жү«жҸҸзҡ„ж—¶й—ҙеҶ…еҚіеҸҜе®ҢжҲҗеҫӘзҺҜ

ж—¶й—ҙеӨҚжқӮеәҰпјҡO(N)

з©әй—ҙеӨҚжқӮеәҰпјҡO(1)

class Solution {
    public String lastSubstring(String s) {
        int itr=1,max=0;
        int k=0;
        while (itr+k<s.length()){
            if(s.charAt(itr+k)==s.charAt(max+k)){
                k++;
            }else if(s.charAt(itr+k)<s.charAt(max+k)){
                itr=itr+k+1;
                k=0;
            }else{
                max=max+k+1;
                if(max>=itr)itr=max+1;
                k=0;
            }
        }
        return s.substring(max,s.length());
    }
}
