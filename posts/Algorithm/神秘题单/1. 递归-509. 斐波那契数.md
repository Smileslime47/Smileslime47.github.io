# 509. 斐波那契数
原题地址：https://leetcode.cn/problems/fibonacci-number/

## 题解
这道题和[70. 爬楼梯](https://leetcode.cn/problems/climbing-stairs/)其实是一样的题目，只是初始化条件有所变化

要注意的是，爬楼梯的i是从1开始的，而斐波那契数列是从0开始的，要考虑到n=0的特殊情况

**时间复杂度**：O(N)

**空间复杂度**：O(N)

```java
class Solution {
    public int fib(int n) {
        if(n==0)return 0;
        int[] fibArr=new int[n+1];
        fibArr[0]=0;
        fibArr[1]=1;
        for(int i=2;i<fibArr.length;i++){
            fibArr[i]=fibArr[i-1]+fibArr[i-2];
        }
        return fibArr[n];
    }
}
```
