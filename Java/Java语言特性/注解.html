жіЁи§ЈжҰӮеҝө #

жіЁи§ЈпјҲAnnotationпјүжҳҜеңЁJDK1.5еҠ е…Ҙзҡ„зү№жҖ§пјҢжҳҜдёҖз§Қзү№ж®Ҡзҡ„жіЁйҮҠпјҢе’ҢжіЁйҮҠдёҚеҗҢзҡ„жҳҜпјҢеңЁзј–иҜ‘ж—¶жіЁи§Јдјҡиў«дёҖеҗҢзј–иҜ‘иҝӣClassж–Үд»¶дёӯпјҢе№¶дё”еҸҜд»Ҙз”ұе…¶д»–зұ»йҖҡиҝҮеҸҚе°„жқҘи°ғз”ЁClassеҜ№иұЎзҡ„жіЁи§ЈеҶ…е®№гҖӮз®ҖеҚ•ең°иҜҙпјҡжіЁи§ЈжҳҜдёҖз§ҚеҸҜд»Ҙиў«е…¶д»–зұ»иҜ»еҸ–зҡ„ж ҮжіЁ

ж— и®әжҳҜеҜ№дәҺзЁӢеәҸе‘ҳиҝҳжҳҜзј–иҜ‘еҷЁпјҢжіЁи§ЈйғҪдёҚдјҡдә§з”ҹд»»дҪ•зҡ„еҪұе“Қпјҡе®ғж—ўдёҚеғҸжіЁйҮҠйӮЈж ·з”ЁдәҺз»ҷзЁӢеәҸе‘ҳж ҮжҳҺд»Јз ҒдҪңз”ЁпјҢд№ҹдёҚеғҸд»Јз ҒйӮЈж ·жҢҮзӨәжңәеҷЁеҰӮдҪ•е·ҘдҪңгҖӮжіЁи§Је”ҜдёҖзҡ„дҪңз”Ёе°ұжҳҜиў«е…¶д»–зұ»иҜ»еҸ–

йҖҡеёёжқҘиҜҙпјҢиҝҷдёӘе…¶д»–зұ»еҲҶдёәдёӨз§Қпјҡзј–иҜ‘еҷЁе’ҢжЎҶжһ¶пјҡ

  • дёҖйғЁеҲҶжіЁи§Јз”ЁдәҺжҢҮзӨәзј–иҜ‘еҷЁе·ҘдҪңпјҢеҰӮ@OverrideгҖҒ@SuppressWarningsзӯү
  • еҸҰдёҖйғЁеҲҶжіЁи§Јз”ЁдәҺжҢҮзӨәжЎҶжһ¶е·ҘдҪңпјҢеҰӮ@ConfigurationгҖҒ@Componentзӯү

JDKжіЁи§Ј #

JDKжң¬иә«жҸҗдҫӣдәҶе°‘ж•°жіЁи§ЈпјҢе…¶дёӯ@SafeVarargsеңЁJDK1.7дёӯеҠ е…ҘпјҢ@FunctionalInterfaceе’Ң@RepeatableеңЁJDK1.8дёӯеҠ е…Ҙ

@Override #

жҢҮзӨәиҜҘж–№жі•йҮҚеҶҷдәҶзҲ¶зұ»ж–№жі•пјҢиӢҘжІЎжңүжӯЈзЎ®йҮҚеҶҷеҲҷдјҡжҠҘй”ҷ

@Deprecated #

жҢҮзӨәиҜҘж–№жі•е·ІиҝҮж—¶пјҢдҪҝз”ЁдјҡиӯҰе‘Ҡ

@SuppressWarnings #

еҝҪи§Ҷзј–иҜ‘еҷЁз»ҷеҮәзҡ„д»»дҪ•иӯҰе‘Ҡ

@SafeVarargs #

еҝҪи§ҶдҪҝз”ЁжіӣеһӢеҸӮж•°зҡ„ж–№жі•/жһ„йҖ еҮҪж•°зҡ„иӯҰе‘Ҡ

@FunctionalInterface #

ж ҮжіЁдёҖдёӘеҮҪж•°ејҸжҺҘеҸЈпјҢеҚід»…еҢ…еҗ«дёҖдёӘж–№жі•зҡ„жҺҘеҸЈ

@Repeatable #

жҢҮзӨәдёҖдёӘжіЁи§ЈеҸҜд»Ҙиў«йҮҚеӨҚеңЁеҗҢдёҖдёӘдҪҚзҪ®дҪҝз”ЁеӨҡж¬Ў

е®ҡд№үжіЁи§Ј #

жҲ‘д»¬еҸҜд»ҘиҮӘе·ұж №жҚ®йңҖжұӮеҲӣе»әж–°зҡ„жіЁи§ЈпјҢйҖҡиҝҮе…ій”®еӯ—@interfaceеҲӣе»әпјҢе№¶з»“еҗҲйғЁеҲҶе…ғжіЁи§ЈиҝӣиЎҢи®ҫе®ҡ

дёҫдҫӢжқҘиҜҙпјҡ

java
@Target(ElementType.TYPE)
@Retention(RetentionPolicy.RUNTIME)
public @interface Report {
    //еҸӮж•°еҲ—иЎЁпјҢе…¶дёӯvalueдёәй»ҳи®ӨеҸӮж•°
    int type() default 0;
    String level() default "info";
    String value() default "";
}

е…ғжіЁи§ЈпјҢеҚідҝ®йҘ°еңЁжіЁи§ЈдёҠзҡ„жіЁи§ЈгҖӮиҝҷзұ»жіЁи§ЈдёҖиҲ¬з”ЁдәҺжҢҮзӨәжіЁи§Јзҡ„еұһжҖ§

@Retention #

жіЁи§Јзҡ„з”ҹе‘Ҫе‘ЁжңҹпјҢеҸӮж•°жңүпјҡжәҗд»Јз ҒпјҲSOURCEпјүгҖҒClassж–Үд»¶пјҲCLASSпјүе’ҢиҝҗиЎҢж—¶пјҲRUNTIMEпјү

@Documented #

жіЁи§ЈжҳҜеҗҰеҢ…еҗ«дәҺиҜҙжҳҺж–ҮжЎЈдёӯ

@Target #

жіЁи§Јзҡ„ж ҮжіЁдҪҚзҪ®пјҢеҸӮж•°жңүпјҡTYPEпјҲзұ»пјү, FIELDпјҲеӯ—ж®өпјү, METHODпјҲж–№жі•пјү, PARAMETERпјҲеҸӮж•°пјү, CONSTRUCTORпјҲжһ„йҖ еҷЁпјү, LOCAL_VARIABLE, ANNOTATION_TYPE, PACKAGE, TYPE_PARAMETER, TYPE_USE

@Inherited #

жҢҮзӨәиҜҘжіЁи§Је…·жңүз»§жүҝжҖ§пјҢеҚідҪҝз”Ё@Inheritedзҡ„жіЁи§Јдјҡе°Ҷзұ»зҡ„иҜҘжіЁи§Јз»§жүҝз»ҷе®ғзҡ„еӯҗзұ»

иҜ»еҸ–жіЁи§Ј #

иҜ»еҸ–жіЁи§Јзҡ„ж–№ејҸе’ҢжіЁи§Јзҡ„з”ҹе‘Ҫе‘Ёжңҹзӣёе…іпјҡ

  • еҜ№дәҺ@RetentionдёәSOURCEзҡ„жіЁи§ЈпјҢеңЁзј–иҜ‘еҗҺиў«дёўеӨұпјҢдёҖиҲ¬еҸӘжңүзј–иҜ‘еҷЁдҪҝз”Ё
  • еҜ№дәҺ@RetentionдёәCLASSзҡ„жіЁи§ЈпјҢдҝқеӯҳеңЁClassж–Үд»¶дёӯпјҢдҪҶеңЁиў«JVMиҜ»еҸ–еҗҺиў«дёўеӨұ
  • еҜ№дәҺ@RetentionдёәRUNTIMEзҡ„жіЁи§ЈпјҢдјҡдҝқеӯҳеңЁClassж–Үд»¶дёӯпјҢе№¶дё”еңЁиҝҗиЎҢж—¶д»ҚеҸҜиў«иҜ»еҸ–

йҖҡиҝҮеҸҚе°„иҜ»еҸ–жіЁи§Ј #

жҲ‘д»¬еҸҜд»ҘйҖҡиҝҮдёҖзі»еҲ—еҸҚе°„зұ»жҸҗдҫӣзҡ„ж–№жі•еҲӨж–ӯдёҖдёӘеҸҚе°„зұ»жҳҜеҗҰеҢ…еҗ«жҹҗжіЁи§ЈпјҢе…¶дёӯеҸӮж•°дј е…ҘеҜ№еә”жіЁи§Јзҡ„ClassеҜ№иұЎ

  • Class.isAnnotationPresent(Class annotation)
  • Field.isAnnotationPresent(Class annotation)
  • Method.isAnnotationPresent(Class annotation)
  • Constructor.isAnnotationPresent(Classannotation)

жӯӨеӨ–иҝҳеҸҜд»ҘйҖҡиҝҮдёҖзі»еҲ—ж–№жі•иҜ»еҸ–дёҖдёӘеҸҚе°„зұ»еҢ…еҗ«зҡ„жіЁи§ЈпјҢе…¶дёӯеҸӮж•°дј е…ҘеҜ№еә”жіЁи§Јзҡ„ClassеҜ№иұЎ

  • Class.getAnnotation(Class annotation)
  • Field.getAnnotation(Class annotation)
  • Method.getAnnotation(Class annotation)
  • Constructor.getAnnotation(Class annotation)

еңЁйҖҡиҝҮдёҠиҝ°ж–№жі•иҺ·еҸ–еҲ°еҸҚе°„зұ»зҡ„жіЁи§Јзҡ„е°ҒиЈ…еҜ№иұЎеҗҺпјҢжҲ‘д»¬е°ұеҸҜд»ҘиҜ»еҸ–жіЁи§Јдёӯзҡ„еҸӮж•°дәҶпјҡ

java
// иҺ·еҸ–Personе®ҡд№үзҡ„@ReportжіЁи§Ј:
Report report = Person.class.getAnnotation(Report.class);
int type = report.type();
String level = report.level();
