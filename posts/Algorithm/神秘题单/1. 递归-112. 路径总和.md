# 112. 路径总和
原题地址：https://leetcode.cn/problems/path-sum/

## 题解
**思路一：深度优先搜索**

DFS会在递归时会从入口节点开始往下遍历子节点，一直搜索到无法递归(return)为止，然后返回到上一层递归

这里我用类似前缀和的思想，用node.val直接存储该节点到根节点的路径和

当满足该节点为叶子节点（左右均为null）且值==targetSum时返回true

**时间复杂度**：O(N)

**空间复杂度**：O(logN)

```java
class Solution {
    boolean flag=false;
    public boolean hasPathSum(TreeNode root, int targetSum) {
        search(root,0,targetSum);
        return flag;
    }

    void search(TreeNode node,int parent,int target){
        if(node == null)return;
        node.val+=parent;
        if(node.val == target&&node.left==null&&node.right==null)flag=true;
        search(node.left,node.val,target);
        search(node.right,node.val,target);
    }
}

```

这是官方题解的**简化版代码**，思路上是相同的

```java
class Solution {
    public boolean hasPathSum(TreeNode root, int targetSum) {
        if(root == null)return false;
        if(root.val == targetSum&&root.left==null&&root.right==null)return true;
        return hasPathSum(root.left,targetSum-root.val)||hasPathSum(root.right,targetSum-root.val);
    }
}
```

**思路二：广度优先搜索**

BFS会在循环时不断将当前节点的子节点加入队列，循环会先将当前层节点遍历完再遍历下一层节点，逐层搜索

同样用一个队列来存储当前节点到根节点的路径和，在将子节点入队时，也计算一次子节点的路径和加入sum队列中

当满足该节点为叶子节点（左右均为null）且sum==targetSum时返回true

**时间复杂度**：O(N)

**空间复杂度**：O(N)

```java
class Solution {
    public boolean hasPathSum(TreeNode root, int targetSum) {
        if(root==null)return false;
        Queue<TreeNode> queueNode=new LinkedList<>();
        Queue<Integer> queueSum=new LinkedList<>();
        queueNode.add(root);
        queueSum.add(root.val);
        TreeNode node;
        int sum;
        while (!queueNode.isEmpty()){
            node=queueNode.remove();
            sum=queueSum.remove();
            if(node.left==null&&node.right==null&&sum==targetSum)return true;
            if(node.left!=null){
                queueNode.add(node.left);
                queueSum.add(sum+node.left.val);
            }
            if(node.right!=null){
                queueNode.add(node.right);
                queueSum.add(sum+node.right.val);
            }
        }
        return false;
    }
}
```
