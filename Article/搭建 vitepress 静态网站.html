жҗӯе»ә vitepress йқҷжҖҒзҪ‘з«ҷ #

Github Repoең°еқҖ

жңҖеҲқеңЁgithubдёҠйғЁзҪІйқҷжҖҒзҪ‘з«ҷжҳҜиҖғиҷ‘еҲ°жң¬дәәеҪ’жЎЈз¬”и®°зҡ„йңҖжұӮпјҡ

зәҝдёӢmarkdownпјҡ

  • жүҖи§ҒеҚіжүҖеҫ—пјҢдё°еҜҢзҡ„жҸ’д»¶ж”ҜжҢҒпјҲvscгҖҒmetionгҖҒtyporaзӯүйғҪжңүеҫҲејәеӨ§зҡ„еҠҹиғҪж”ҜжҢҒ/жҸ’д»¶зӨҫеҢәпјү
  • ж— жі•еӨҡи®ҫеӨҮеҗҢжӯҘпјҢзј–иҫ‘еӨҡдәҶдјҡеҮәзҺ°еҶ…е®№еҶІзӘҒ

еҗҢжӯҘеҲ°githubпјҡ

  • е®һзҺ°дәҶеҗҢжӯҘпјҢrepoдёҠзҡ„mdж–Үд»¶еҚідёәеҗҢжӯҘзүҲжң¬
  • еӨұеҺ»дәҶзәҝдёӢзј–иҫ‘еҷЁејәеӨ§зҡ„еҠҹиғҪж”ҜжҢҒпјҢдҫӢеҰӮиҷҪз„¶githubзҡ„markdownйў„и§Ҳиҝҳж”ҜжҢҒmermaidжөҒзЁӢеӣҫжёІжҹ“пјҢдҪҶжҳҜеҚҙж— жі•жӯЈеёёйў„и§Ҳlatexе…¬ејҸ

еңЁеҸҢйҮҚйңҖжұӮдёӢпјҢеңЁgithubдёҠйғЁзҪІдёҖдёӘеҹәдәҺvueзҡ„йқҷжҖҒзҪ‘з«ҷдҫҝжҲҗдёәдәҶдҪҺжҲҗжң¬дёӢзҡ„жңҖдјҳи§Ј

дёҺvuepressзӣёжҜ” #

vitepressзӣёжҜ”vuepressжңүзқҖиҪ»йҮҸеҢ–пјҢз»“жһ„з®ҖжҙҒпјҢй…ҚзҪ®з®ҖеҚ•пјҢйҮҮз”Ёvue3е’Ңviteзҡ„дјҳеҠҝпјҢдҪҶжҳҜзӣёжҜ”д№ӢдёӢзӨҫеҢәе’ҢжҸ’д»¶иҝңжІЎжңүvuepressеәһеӨ§пјҲж–ҮжЎЈдёӯжҸҗеҲ°иҝҮдёҚжү“з®—е°Ҷе…¶еҒҡжҲҗвҖңеҸҰдёҖдёӘvue pressвҖқпјү

жҗӯе»ә #

Vitepressе®ҳж–№ж–ҮжЎЈдёӯе·Із»ҸеҫҲиҜҰз»Ҷең°иҜҙжҳҺдәҶжҗӯе»әжөҒзЁӢпјҢжӯӨеӨ–д№ҹеҸҜд»ҘеҸӮз…§иҝҷзҜҮдёӯж–Үж–ҮжЎЈ

иҝҷйҮҢд»…дёәе®ҳж–№ж–ҮжЎЈзҡ„еҶҚжҖ»з»“
