и®ҫdp[i]дёәsзҡ„еӯҗдёІs[0,i]зҡ„ж–№жЎҲж•°

д»ӨjйҒҚеҺҶi~0пјҢиҮӘз„¶ең°пјҢжҲ‘д»¬еҸҜд»Ҙе°ҶеӯҗдёІиҝӣдёҖжӯҘеҲҮеҲҶдёәs[0,j-1]е’Ң[j,i]дёӨйғЁеҲҶ

  • еҗҢж—¶пјҢ[j,i]иҝҷйғЁеҲҶеӯҗдёІжҲ‘д»¬еҸҜд»Ҙе°Ҷе…¶и§ЈжһҗдёәдёҖдёӘж–°зҡ„ж•°еӯ—num
  • жӯӨж—¶пјҢиӢҘиҝҷдёӘи§ЈжһҗеҮәжқҘзҡ„ж•°еӯ—ж»Ўи¶іжқЎд»¶пјҲеҚіпјүпјҢеҲҷsзҡ„еӯҗдёІs[0,i]зҡ„ж–№жЎҲж•°еҸҳжҲҗдәҶsзҡ„еӯҗдёІs[0,j-1]зҡ„ж–№жЎҲж•°иҝҷдёӘеӯҗй—®йўҳ

жӯӨж—¶жҲ‘д»¬еҫ—еҲ°дәҶдёҖдёӘеҹәжң¬зҡ„DPе…¬ејҸпјҡ

еҜ№дәҺеӯҗдёІnum[j,i]зҡ„и§Јжһҗж•°еӯ—пјҡ

еҜ№дәҺд»»ж„Ҹjж»Ўи¶іпјҡ

иҖғиҷ‘еҲ°еҮ з§Қзү№ж®Ҡжғ…еҶөпјҡ

  • и§Јжһҗж•°еӯ—зҡ„иҢғеӣҙеңЁ[1,k]пјҢжүҖд»Ҙж•°еӯ—дёә0ж—¶еҸҜд»ҘзӣҙжҺҘcontinue
  • и§Јжһҗж•°еӯ—дёҚеҢ…еҗ«еүҚеҜј0пјҢжүҖд»Ҙs[j]==0ж—¶еҸҜд»ҘзӣҙжҺҘcontinue
  • и§Јжһҗж•°еӯ—еҝ…е®ҡе°ҸдәҺзӯүдәҺkпјҢеҪ“и§Јжһҗж•°еӯ—зҡ„дҪҚж•°еӨ§дәҺkзҡ„дҪҚж•°жҲ–иҖ…и§Јжһҗж•°еӯ—еӨ§дәҺkж—¶пјҢеҝ…е®ҡдёҚеӯҳеңЁи§ЈпјҢеҸҜд»ҘзӣҙжҺҘbreak

еҜ№дәҺиҫ№з•Ңжғ…еҶөпјҢеҪ“jдёә0ж—¶пјҢиӢҘеӯҗдёІ[0,i]зҡ„и§Јжһҗж•°еӯ—ж»Ўи¶іжғ…еҶөпјҢжӯӨж—¶дёҚеӯҳеңЁdp[j-1]пјҢж–№жЎҲж•°еә”дёә1пјҢеҚі

жңҖеҗҺдёҚиҰҒеҝҳдәҶеҜ№dp[i]%=1e9+7

ж—¶й—ҙеӨҚжқӮеәҰпјҡO(N^2)

з©әй—ҙеӨҚжқӮеәҰпјҡO(N)

class Solution {
    public int numberOfArrays(String s, int k) {
        int digit=1,kcopy=k;
        while(kcopy>=10){
            kcopy/=10;
            digit++;
        }

        long[] dp=new long[s.length()];
        long num,pow;
        for(int i=0;i<s.length();i++){
            num=0;
            pow=1;
            for(int j=i;j>=0;j--){
                if(i-j+1>digit)break;
                num=num+(s.charAt(j)-'0')*pow;
                pow*=10;
                if(s.charAt(j)=='0'||num==0){
                    continue;
                }
                if(num<=k){
                    dp[i]+=j-1>=0?dp[j-1]:1;
                    dp[i]%=1000000007;
                }else break;
            }
        }
        return (int)(dp[s.length()-1]);
    }
}
